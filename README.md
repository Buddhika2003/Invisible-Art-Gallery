# Portrait Art Website

A custom portrait art website where customers can view artworks, request personalized portraits, and communicate directly with the artist.  
Built with HTML, CSS, JavaScript, PHP, and MySQL designed and coded by **Buddhika Rathnayake**.

---

# Project Structure

portrait-art/
├── index.php # Home page
├── gallery.php # Artworks gallery
├── order.php # Order/custom portrait form
├── submit_order.php # Form backend handler
├── db_connect.php # MySQL database connection
├── includes/ # Header & footer components
├── admin/ # Admin login and dashboard
├── assets/ # CSS, JS, and images
└── upload/ # Uploaded customer photos
